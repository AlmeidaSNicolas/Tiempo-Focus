# ⏱️ PomoNs - React Pomodoro Timer

Este projeto é um Timer de Pomodoro desenvolvido durante o curso de **React JS e TypeScript** do professor Luiz Otávio Miranda na Udemy. A aplicação foca na gestão de tempo e produtividade, aplicando conceitos avançados de estado e hooks no React.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

* **React JS**: Biblioteca principal para a interface.
* **TypeScript**: Tipagem estática para maior segurança e produtividade.
* **CSS Modules**: Para estilização escopada e organizada.
* **Lucide React / Icons**: (Caso tenha usado ícones específicos).

## 🧠 Conceitos Praticados

Durante o desenvolvimento, foram aplicados conceitos fundamentais de desenvolvimento web moderno:

* **Hooks**: `useState`, `useEffect` e `useCallback`.
* **Gerenciamento de Ciclo de Vida**: Controle preciso do tempo e intervalos (setInterval).
* **Componentização**: Divisão da interface em partes lógicas e reutilizáveis.
* **Tipagem de Props e Eventos**: Uso rigoroso de interfaces e tipos com TypeScript.

## 🛠️ Como executar o projeto

1.  Clone o repositório:
    ```bash
    git clone [https://github.com/AlmeidaSNicolas/PomoNs.git](https://github.com/AlmeidaSNicolas/PomoNs.git)
    ```
2.  Entre na pasta do projeto:
    ```bash
    cd PomoNs/pomodoro
    ```
3.  Instale as dependências:
    ```bash
    npm install
    ```
4.  Inicie a aplicação:
    ```bash
    npm start
    ```
